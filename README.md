# Customer_Behaviour_Analyst
End-to-end data analytics project analyzing customer behavior using Python, SQL, and Power BI with EDA, data wrangling, and interactive dashboard visualization

## Project Overview
This project analyzes customer shopping behavior using transactional data to uncover insights about spending patterns, customer segments, product preferences, and subscription behavior.  

The goal is to help businesses make **data-driven decisions** to improve marketing strategies, customer engagement, and product positioning.

Dataset: 3,900 purchase transactions with 18 features including demographics, purchase details, and behavioral indicators.

---

## Business Problem
A retail company wants to better understand how customers interact with their products and services.

Key question:
**How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing strategies?** :contentReference[oaicite:1]{index=1}

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Power BI
- Data Cleaning & Wrangling
- Exploratory Data Analysis (EDA)

---

## Project Workflow

### 1. Data Preparation (Python)
- Loaded dataset using Pandas
- Handled missing values in review ratings
- Standardized column names
- Created new features such as `age_group`
- Performed exploratory data analysis

### 2. Data Analysis (SQL)
Key business questions answered:
- Revenue comparison between male vs female customers
- Customers who used discounts but spent above average
- Top-rated products
- Subscription vs non-subscription spending
- Customer segmentation (New, Returning, Loyal)
- Revenue contribution by age group

### 3. Data Visualization (Power BI)
Built an **interactive dashboard** showing:
- Total customers
- Average purchase amount
- Product category sales
- Customer segmentation
- Revenue by age group
- Subscription distribution

---

## Key Insights
- Loyal customers contribute the majority of purchases.
- Some discounted products drive a large portion of sales.
- Certain age groups contribute higher revenue.
- Express shipping customers show slightly higher purchase value.

---

## Business Recommendations
- Promote **subscription benefits** to increase recurring revenue.
- Implement **customer loyalty programs** for repeat buyers.
- Highlight **top-rated products** in marketing campaigns.
- Use **targeted marketing for high-value customer segments**.
